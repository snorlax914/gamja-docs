## 1. 최종 발표 준비 및 산출물 정리

이번 주는 신규 기능 추가 없이 **최종 발표 준비**에 집중했다. 6주차까지 구현한 MVP가 안정적으로 동작하는 상태이므로, 발표 품질을 높이는 데 리소스를 집중하는 것이 적절하다고 판단했다.

### 1.1 주차보고서 정리

- 5~7주차 보고서 형식 통일 및 누락 항목 보완
- 최종 제출을 위한 문서 구조 정리

### 1.2 결과보고서 작성

- 프로젝트 전체 결과를 요약한 최종 결과보고서 작성
- 목표 달성 여부, 기술 스택, 산출물 목록 등 포함

## 2. 향후 확장 가능성

서비스 확장 시 고려할 수 있는 기능을 정리한다.

| 항목 | 설명 | 비고 |
|---|---|---|
| 카테고리별 채팅 | 카테고리 단위로 문서를 필터링한 후 RAG 질의 | Qdrant 필터 확장으로 구현 가능 |
| 전체 문서 대상 채팅 | 특정 문서가 아닌 전체 문서 기반 RAG | Qdrant 필터 조건 제거로 대응 가능 |
